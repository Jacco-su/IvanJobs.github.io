在调研技术选型的时候，开源项目的代码协议是我们比较关注的点，所以有必要总结下来进行记忆。

![](/assets/free_software_licenses.png)

### BSD
可以为所欲为的协议！

### Apache 2.0
类似BSD, 商业友好协议。

### GPL
商业不友好，不允许修改后和衍生的代码作为闭源的商业软件发布和销售。

### LGPL
允许商业软件通过类库引用的方式使用

### MIT
商业友好协议。
